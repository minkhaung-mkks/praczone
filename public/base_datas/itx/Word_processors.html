1. Overview

A word processing application is software that allows you to create, open, and update digital documents. The two most popular word processors are:
  • Microsoft Word (part of Office 365): Uses .doc (older versions) and .docx (newer versions) extensions.
  • Google Docs: Browser-based and free, accessible with a Gmail account.

Key Features:

  • Text Formatting: Colors, fonts, line spacing, special fonts, hyperlinks, and references.
  • Collaboration Tools: Track changes, comments, version comparison.
  • Templates: Preformatted templates for various document types (invitations, resumes, brochures).
  • Page Layout: Margins, orientation, headers, footers, page numbering, table of contents.
  • Export Options: Save in multiple formats like PDF, HTML, EPUB.
  • Revision Tools: Spell check, grammar check, thesaurus, find and replace.
  • Insert Options: Images, shapes, tables, hyperlinks, bookmarks, footnotes.

2. Getting Started

Launching Microsoft Word:

  1. From Start Menu:
    • Click on the Microsoft Start button.
    • Type “Word” in the search box.
    • Click the Word application icon to open.
  2. Opening an Existing Document:
    • Double-click on a .doc or .docx file.
    • The operating system uses file associations to open Word and display the document.

Launching Google Docs:

  1. Via URL:
    • Type https://docs.google.com in your browser’s address bar.
  2. Via Gmail:
    • Open Gmail.
    • Click on the dots next to your name in the top right corner.
    • Select Docs from the list of apps.

Using Templates:

  • Accessing Templates:
    • Both Microsoft Word and Google Docs offer preformatted templates for documents like invitations, resumes, certificates, and brochures.
    • Benefits: Saves time, ensures professional appearance, includes essential elements.

3. Essential File Operations

Word processing applications provide essential operations for saving and sharing documents.

Save

  • Initial Save:
    • Save your document with a meaningful name and in a suitable location.
  • Auto Save:
    • Microsoft Word: Integrates with OneDrive for automatic saving.
    • Google Docs: Automatically saves to Google Drive as you work.
  • Manual Save Shortcut: Ctrl + S
  • Benefits:
    • Protects against data loss from power outages or computer issues.
    • Saves space locally or on the cloud based on settings.

Print

  • Accessing Print Options:
    • Microsoft Word: File > Print or Ctrl + P
    • Google Docs: File > Print or Ctrl + P
  • Print Dialog Options:
    • Number of pages, specific pages, number of copies.
    • Paper size, margins, single/double-sided printing.
    • Print Preview to check layout before printing.

Export

  • Export Formats:
    • PDF: Read-only format for easy sharing.
    • HTML: Save as a webpage.
    • EPUB: Electronic Book format for e-readers like Kindle or Nook.
  • Exporting in Google Docs:
    • File > Download > choose format (e.g., .docx, .pdf, .html, .epub)
  • Exporting in Microsoft Word:
    • File > Save As > choose format.

4. Creating Content

Creating content in word processors involves various formatting and organizational tools to enhance readability and professionalism.

Fonts

Fonts refer to the style, shape, and size of text characters. They enhance readability and convey emotions.

Categories of Fonts:

  1. Serif Fonts:
    • Features: Small decorative lines (serifs) at the ends of letters.
    • Perception: Formal, authoritative, mature.
    • Examples: Times New Roman, Georgia, Courier.
    • Usage: Print publications like magazines and novels.
  2. Sans-Serif Fonts:
    • Features: No serifs; clean and modern look.
    • Perception: Modern, easy to read on screens.
    • Examples: Arial, Calibri, Verdana.
    • Usage: Digital publications, online documents.
  3. Script Fonts:
    • Features: Mimic handwritten text; often decorative.
    • Perception: Personalized, elegant.
    • Examples: Segoe Script, Lucinda Handwriting, Brush Script.
    • Usage: Invitations, certificates, personalized documents.

Points and Pixels:

  • Point (pt): 1 pt = 1/72 inch.
  • Pixel (px): 1 px = 1/96 inch.
  • Standard Conversion: 12 pt ≈ 16 px.
  • Recommendation: Font size between 11–12 points for readability.

Spacing

  • Line Spacing: Distance between lines within a paragraph.
  • Paragraph Spacing: Distance between separate paragraphs.
  • Recommendations:
    • Line Spacing: At least single-spaced.
    • Paragraph Spacing: Add 6 points between paragraphs for clarity.

Indentation and Alignment

  • Indentation: Space between the margin and the beginning of text.
    • First Line Indent: Indents the first line of a paragraph.
    • Hanging Indent: Indents all lines except the first in a paragraph.
  • Alignment:
    • Left-Aligned: Flush with the left margin.
    • Right-Aligned: Flush with the right margin.
    • Centered: Equidistant from both margins.
    • Justified: Flush with both margins, adding space between words as needed.

Lists

  • Bulleted Lists:
    • Use: Items in no specific order.
    • Example:
      • Animals
        • Cat
        • Dog
        • Horse
  • Numbered Lists:
    • Use: Items in a specific sequence.
    • Example:
      1. First step
      2. Second step
      3. Third step
  • Nested Lists: Lists within lists for hierarchical information.

Styles

Styles are predefined combinations of formatting features that can be applied to text to ensure consistency and save time.
  • Benefits:
    • Consistent look and feel across the document.
    • Easy global changes by modifying the style definition.
    • Hierarchical styles allow for dependent changes.
  • Common Styles:
    • Headings: Heading 1, Heading 2, etc., used for sections and chapters.
    • Body Text: Standard text formatting.
    • Captions: Titles or explanations for tables and figures.

Tables

  • Purpose: Organize complex data, highlight trends, and improve readability.
  • Features:
    • Insert, format, and edit tables easily.
    • Use predefined table templates.
    • Add or remove rows and columns.
Example Table:
Role Coastside High Bayside High
Quarterback Alex Johnston Conan Morrison
Left Tackle Angel Howard Daryl Gilliam
Left Guard Arthur Smith Edmund Day
Center Ben Knight Harold Sloan
Quarterback Ryan Carter Jack Walker

Captions

  • Definition: Titles or brief explanations added to tables, figures, and other objects.
  • Features:
    • Automatic sequence numbering.
    • Distinctive appearance through style application.
    • Referencing within the document.
Inserting Captions:
  • Microsoft Word:
    • References > Insert Caption
  • Google Docs:
    • Insert a caption manually as Google Docs doesn’t have a direct caption feature.

References

  • Purpose: Enhance credibility, allow readers to locate related information.
  • Types:
    • Citations: References to sources of information.
    • Bibliography: List of all sources used.
    • Hyperlinks: Shortcuts to other parts of the document or external websites.
    • Bookmarks: Mark specific locations in the document for easy navigation.
    • Footnotes: Additional information at the bottom of the page.
    • Table of Contents: Organized list of chapters and sections with page numbers.

5. Revision and Proofreading

Word processors offer powerful tools to help improve your writing and correct errors.

Track Changes

  • Purpose: Keep track of edits and changes made by multiple reviewers.
  • Features:
    • Highlights additions, deletions, and modifications.
    • Allows accepting or rejecting changes.
    • Different colors for different authors.
Inserting Track Changes:
  • Microsoft Word:
    • Review > Track Changes or Ctrl + Shift + E
  • Google Docs:
    • Editing dropdown: Select "Suggesting" mode.

Comments

  • Purpose: Add remarks or observations without altering the main text.
  • Usage:
    • Suggest adding examples, simplifying language, reducing length, etc.
Inserting Comments:
  • Microsoft Word:
    • Insert > Comment or highlight text and right-click to select "Comment"
  • Google Docs:
    • Highlight text, click "Add a comment" in the toolbar.

Find and Replace

  • Purpose: Quickly locate and replace words or phrases in the document.
  • Features:
    • Replace single occurrences or all at once.
    • Advanced options for finding variations (e.g., dance, danced, dances, dancing).
Using Find and Replace:
  • Microsoft Word:
    • Ctrl + F for Find, Ctrl + H for Find and Replace
  • Google Docs:
    • Edit > Find and Replace

Spelling and Grammar

  • Purpose: Identify and correct spelling and grammatical errors.
  • Features:
    • In-built spell check and grammar suggestions.
    • Add new words to the dictionary.
Using Spelling and Grammar:
  • Microsoft Word:
    • Review > Spelling & Grammar
  • Google Docs:
    • Tools > Spelling and grammar

Thesaurus

  • Purpose: Find synonyms to avoid repetition and enhance vocabulary.
  • Usage:
    • Replace overused words with alternatives.
Using Thesaurus:
  • Microsoft Word:
    • Review > Thesaurus
  • Google Docs:
    • Tools > Dictionary

6. References

Citations and Bibliography

  • Purpose: Credit sources and allow readers to locate them.
  • Features:
    • Industry-standard formats (MLA, APA, Chicago).
    • Automatically generate and update bibliography.
Creating Citations:
  • Microsoft Word:
    • References > Insert Citation
  • Google Docs:
    • Tools > Citations

Hyperlinks

  • Purpose: Link to external websites or different sections within the document.
  • Usage:
    • Enhance navigation and provide additional resources.
Inserting Hyperlinks:
  • Microsoft Word:
    • Highlight text, right-click, select "Link", enter URL or document section.
  • Google Docs:
    • Highlight text, click "Insert link" icon or Ctrl + K, enter URL or document section.

Bookmarks

  • Purpose: Mark specific locations in the document for easy navigation.
  • Usage: Create links to bookmarks for internal referencing.
Inserting Bookmarks:
  • Microsoft Word:
    • Insert > Bookmark
  • Google Docs:
    • Use headings or manually add bookmarks via Insert > Bookmark

Footnotes

  • Purpose: Provide additional information without disrupting the main text flow.
  • Usage: Explain terms, cite sources, add context.
Inserting Footnotes:
  • Microsoft Word:
    • References > Insert Footnote
  • Google Docs:
    • Insert > Footnote

Table of Contents

  • Purpose: Provide an organized list of chapters and sections with page numbers.
  • Features:
    • Automatically generated based on heading styles.
    • Live updates as the document changes.
Inserting Table of Contents:
  • Microsoft Word:
    • References > Table of Contents
  • Google Docs:
    • Insert > Table of contents
Keyboard Shortcut to Update Fields and TOC:
  1. Select entire document: Ctrl + A

7. Mail Merge

Mail Merge allows you to create personalized documents or emails for multiple recipients by combining a template with a data source.

Steps to Perform Mail Merge in Microsoft Word:

  1. Create the Template:
    • Write the letter or email in a Word document.
    • Example: Holiday greetings, promotional letters, grade letters.
  2. Prepare the Recipient List:
    • Use an Excel sheet with a header row (e.g., First Name, Last Name, Email).
    • Each row represents a recipient.
  3. Link the Data Source:
    • In Word, go to Mailings > Select Recipients > Use an Existing List.
    • Choose the Excel file with recipients.
  4. Insert Merge Fields:
    • Mailings > Insert Merge Field to add personalized fields (e.g., First Name).
  5. Preview Results:
    • Mailings > Preview Results to see how the merged documents will look.
  6. Finish & Merge:
    • Mailings > Finish & Merge > Print Documents or Send Email Messages.

Benefits of Mail Merge:

  • Efficiency: Create thousands of personalized documents quickly.
  • Accuracy: Reduces the risk of errors from manual entry.
  • Personalization: Each document is tailored to the recipient.

9. Glossary of Terms

  • Word Processor: Software application for creating, editing, and formatting text documents.
  • Template: Preformatted document structure that serves as a starting point for new documents.
  • Font: A set of characters with a specific style, size, and weight.
  • Serif: Decorative lines at the ends of characters in certain fonts.
  • Sans-Serif: Fonts without decorative lines at the ends of characters.
  • Script Font: Fonts that mimic handwritten or cursive styles.
  • Point (pt): Unit of measurement for font size; 1 pt = 1/72 inch.
  • Pixel (px): Unit of measurement for digital displays; 1 px = 1/96 inch.
  • Indentation: Space between the margin and the beginning of a paragraph.
  • Alignment: Orientation of text within a document (left, right, center, justified).
  • Bulleted List: A list where items are marked with bullets.
  • Numbered List: A list where items are numbered in sequence.
  • Styles: Predefined formatting settings applied to text for consistency.
  • Table: A grid of rows and columns used to organize data.
  • Caption: A title or explanation added to tables, figures, etc.
  • Citation: A reference to a source of information.
  • Bibliography: A list of all sources referenced in a document.
  • Hyperlink: A clickable link that directs to another location in the document or an external website.
  • Bookmark: A marker in the document for easy navigation.
  • Footnote: Additional information placed at the bottom of a page.
  • Table of Contents (TOC): An organized list of document sections with page numbers.
  • Track Changes: A feature that records edits made to a document.
  • Comments: Notes or remarks added to a document without altering the main text.
  • Mail Merge: A process to create personalized documents for multiple recipients using a template and data source.
