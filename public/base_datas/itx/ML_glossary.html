Glossary of Terms

Term Definition
Activation Function A mathematical function applied to a neuron's output in a neural network to introduce non-linearity, allowing the network to learn complex patterns. Examples include Sigmoid, Tanh, and ReLU functions.
Artificial Intelligence (AI) The broader field of creating machines capable of performing tasks that typically require human intelligence. This includes learning, reasoning, problem-solving, perception, and language understanding.
Bias In machine learning, bias refers to the error introduced by approximating a real-world problem, which may be extremely complex, by a much simpler model. High bias can lead to underfitting.
Clustering An unsupervised learning task of grouping a set of objects so that objects in the same group (cluster) are more similar to each other than to those in other groups.
Deep Learning A subset of machine learning involving neural networks with multiple layers (deep architectures) that can learn hierarchical representations of data.
Epoch One complete pass through the entire training dataset during the training process of a machine learning model.
Feature An individual measurable property or characteristic used as an input in a machine learning model. Also known as a variable or attribute.
k-Nearest Neighbors (k-NN) A non-parametric method used for classification and regression by comparing a data point with its 'k' nearest neighbors in the feature space and making decisions based on majority class or average value.
Learning Rate A hyperparameter that controls how much we adjust the weights of our network with respect to the gradient during training. It determines the step size at each iteration while moving toward a minimum of the loss function.
Loss Function A function that measures the difference between the predicted output of a model and the actual output. It guides the optimization process by indicating how well the model is performing.
Machine Learning (ML) A branch of artificial intelligence focused on building systems that learn from data to improve performance on a specific task without being explicitly programmed.
Neural Network A series of algorithms that mimic the operations of a human brain to recognize relationships between vast amounts of data through interconnected nodes (neurons) in layered structures.
Overfitting A modeling error which occurs when a function is too closely fit to a limited set of data points, capturing noise along with the underlying pattern. This results in poor performance on new, unseen data.
Perceptron The simplest type of artificial neural network, consisting of a single neuron with adjustable weights and bias. Used for binary classification tasks.
Reinforcement Learning A type of machine learning where an agent learns to make decisions by performing actions and receiving feedback through rewards or penalties, aiming to maximize cumulative rewards over time.
Supervised Learning A type of machine learning where the model is trained on labeled data, learning to map inputs to outputs based on example input-output pairs.
Unsupervised Learning A type of machine learning where the model is trained on unlabeled data, aiming to find hidden patterns or intrinsic structures without explicit instructions on what to predict.
Weight In neural networks, a parameter that transforms input data within the network's layers. Weights are adjusted during training to minimize the loss function.
Activation Function A mathematical function that determines the output of a neural network's node given an input or set of inputs. Activation functions introduce non-linearities into the network, enabling it to learn complex patterns.
Hyperparameter A configuration that is external to the model and whose value cannot be estimated from data. Hyperparameters must be set before the learning process begins. Examples include learning rate, number of epochs, and batch size.
Parameter An internal variable of a model that is learned from the training data, such as weights and biases in neural networks.
Feature Engineering The process of using domain knowledge to extract features from raw data to improve the performance of machine learning models. This involves creating new features or transforming existing ones.
Underfitting Occurs when a machine learning model cannot capture the underlying pattern of the data, often due to its simplicity. This results in poor performance on both training and new data.
Generalization The ability of a machine learning model to perform well on new, unseen data, not just the data it was trained on. A model that generalizes well can make accurate predictions on novel inputs.
Regularization Techniques used to reduce overfitting by discouraging overly complex models through the addition of a penalty term to the loss function. Examples include L1 and L2 regularization.
Cross-Validation A statistical method used to estimate the skill of machine learning models. It involves partitioning the data into complementary subsets, performing training and validation on them, and averaging the results.
Decision Boundary A hypersurface that partitions the underlying vector space into two sets, one for each class. In classification, the decision boundary separates different classes.
Gradient Descent An optimization algorithm used to minimize the loss function by iteratively moving in the direction of steepest descent as defined by the negative of the gradient.
Backpropagation A training algorithm for neural networks that calculates the gradient of the loss function with respect to the weights, allowing for efficient computation of weight updates.
Dropout A regularization technique where randomly selected neurons are ignored during training, which prevents units from co-adapting too much and reduces overfitting.
Batch Size The number of training examples used in one iteration of model parameter updates during training. Smaller batch sizes can lead to noisier updates but can improve generalization.
Stochastic Gradient Descent An optimization method that updates model parameters using a single or a few training examples at a time, introducing randomness and often speeding up convergence.
Confusion Matrix A table used to describe the performance of a classification model by comparing predicted and actual classes, showing true positives, false positives, true negatives, and false negatives.
Precision and Recall Metrics used to evaluate the performance of classification models. Precision is the ratio of true positives to all predicted positives, while recall is the ratio of true positives to all actual positives.
F1 Score The harmonic mean of precision and recall, providing a single metric that balances both concerns in classification tasks.
ROC Curve Receiver Operating Characteristic curve; a graphical plot illustrating the diagnostic ability of a binary classifier as its discrimination threshold is varied, plotting true positive rate against false positive rate.
AUC Area Under the ROC Curve; a measure of the ability of a classifier to distinguish between classes, with higher values indicating better performance.
Hyperplane In machine learning, especially SVMs, a hyperplane is a decision boundary that separates different classes in a feature space.
Kernel Trick A method used in machine learning algorithms, like SVMs, to operate in a high-dimensional implicit feature space without computing the coordinates of the data in that space, but by computing the inner products between the images of all pairs of data in the feature space.
Support Vector Machine (SVM) A supervised learning algorithm used for classification and regression tasks that finds the hyperplane that best separates different classes by maximizing the margin between data points of different classes.
Principal Component Analysis (PCA) A dimensionality reduction technique that transforms data into a new coordinate system, reducing the number of variables while retaining the most important variance features by projecting data onto principal components (eigenvectors).
Dimensionality Reduction Techniques used to reduce the number of input variables in a dataset, simplifying models and helping to visualize data in lower dimensions.
Data Augmentation Techniques used to increase the diversity of training data by applying random transformations, such as rotations or flips, to existing data, improving model robustness and generalization.
Transfer Learning A machine learning method where a model developed for one task is reused as the starting point for a model on a second task, leveraging learned features and reducing training time.
